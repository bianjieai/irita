| 12     | sdk      | 交易备注超出长度限制 |
| 13     | sdk      | 交易费用不足         |
| 14     | sdk      | 超出最大签名数       |
| 15     | sdk      | 签名缺失             |
| 16     | sdk      | JSON 序列化失败      |
| 17     | sdk      | JSON 反序列化失败    |
